# GhostEscape
**GhostEscape** is a cross-platform survival game developed in C++ with SDL3 and glm.

## Control
```
W,A,S,D - to move;
Mouse Left - to attack;
Mouse Right - slow motion;
F4 - toggle fullscreen;
```
